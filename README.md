## How to use

Make sure all the dependency requirements are installed!

```bash
# Clone this repository
$ git clone https://github.com/openpmap

# Go into the repository
$ cd open-pmap

# Build the project
$ mvn clean install

# Run the jar
$ java -jar openpmap.jar
```
